## Data Source

The dataset used in this project is the Online Retail II Dataset, which contains transactional data for a UK-based online retail company.

Due to file size limitations, the dataset is not included in this repository.

You can download the dataset from:
https://archive.ics.uci.edu/ml/datasets/Online+Retail+II

After downloading, place the file in the data/ folder to run the notebook locally.
